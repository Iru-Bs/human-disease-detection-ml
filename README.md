# Detection and Prediction of Human Common Diseases using Machine Learning

This project focuses on detecting and predicting common human diseases using machine learning techniques. The system uses image data to classify whether a person is healthy or affected by a disease.

## Project Description
The model is developed using a Convolutional Neural Network (CNN) architecture, specifically MobileNetV2, to perform disease classification. A web application is created using Flask where users can upload images and receive predictions about possible diseases.

## Features
- Detects common human diseases using image input
- Classifies images as Healthy or Diseased
- Displays prediction results with confidence level
- User-friendly web interface for uploading images

## Technologies Used
- Python
- Machine Learning
- Deep Learning (CNN – MobileNetV2)
- Flask
- NumPy
- Pandas
- Matplotlib

## Application
This system can help in early disease detection and assist users in identifying possible health conditions through image-based analysis.
