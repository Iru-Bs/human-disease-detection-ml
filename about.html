About This Project

The Human Disease Detection System is an AI-powered web application designed to assist in the early detection of diseases from human facial or skin images. By leveraging deep learning and computer vision, it provides accurate predictions to help users assess their health status instantly.

🧬 Project Overview

This system uses a Convolutional Neural Network (CNN) model based on MobileNetV2 — a lightweight and efficient deep learning architecture. The model has been trained on labeled image data covering multiple conditions such as:

  • Allergy
  • Cold
  • Healthy
  • Infection
  • Jaundice
  • Malaria
  • Skin Infection
  • Smallpox

⚙️ Technologies Used

  • Frontend: HTML, CSS, Bootstrap 5
  • Backend: Flask (Python)
  • Machine Learning: TensorFlow / Keras (MobileNetV2)
  • Image Handling: Pillow (PIL)

🎯 Objective

To create a user-friendly and intelligent platform that can quickly identify potential diseases using AI, reducing manual diagnosis time and assisting medical professionals and individuals alike.

🚀 Future Scope

  • Integration with mobile health monitoring systems
  • Real-time webcam or video-based diagnosis
  • Expanded dataset for more disease categories
  • Connecting with medical professionals for consultation
